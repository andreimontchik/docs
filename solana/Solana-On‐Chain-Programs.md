# Info

- Program keys
  - **Program Id** - the on-chain program identifier. Generated by the `solana program deploy` command.
  - **Owner** - The [BPF Loader](https://docs.solanalabs.com/runtime/programs#bpf-loader) program id.
  - **ProgramData Address** -
  - **Upgrade Authority** - pubkey of the `solana program deploy --upgrade-authority` parameter. Default pubkey is used if not provided.

# [Create](https://solana.com/developers/guides/getstarted/local-rust-hello-world#create-your-first-solana-program)

1. Create new Rust library: `cargo init <NAME> --lib`.
1. Update the Cargo.toml:

   ```
   [package]
   name = "<NAME>"
   version = "0.1.0"
   edition = "2021"

   # See more keys and their definitions at https://doc.rust-lang.org/cargo/reference/manifest.html

   [dependencies]
   solana-program = "=1.17.31"

   [lib]
   crate-type = ["cdylib", "lib"]
   ```

1. Create the program keypair: `solana-keygen new -o <PROGRAM NAME>.json`
1. Use the [Solana Developers Program Examples](https://solana.com/docs/programs/examples) and [SPL Examples](https://github.com/solana-labs/solana-program-library/tree/master/examples/rust) for additional details.
   - [Hello Solana](https://github.com/solana-developers/program-examples/tree/main/basics/hello-solana)
   - [Custom instructions](https://github.com/solana-developers/program-examples/tree/main/basics/processing-instructions)
     [Transactions](Solana-Transactions.md)
1. [Cross-program Invocation aka CPI](https://solana.com/docs/core/cpi)
   - [CPI in Solana Cookbook](https://solanacookbook.com/references/programs.html#how-to-do-cross-program-invocation)
   - [CPI program examples](https://github.com/solana-developers/program-examples/tree/main/basics/cross-program-invocation)
   - [How to CPI wiht PDA Signer](https://solana.com/developers/guides/getstarted/how-to-cpi-with-signer)

# [Build](https://solana.com/developers/guides/getstarted/local-rust-hello-world#build-your-rust-program)

1. Run the `cargo build-bpf`, this will build program binary as well as the keypair file containing program id in the `target/deploy/<CRATE NAME>` directory.

   > In case if compilation fails complaining about an obsolete Rust version, add the Solana platform-tools version parameter to the command `cargo build-bpf --tools-version v1.39`. Versions are tagged in the [platform-tools](https://github.com/anza-xyz/platform-tools/tags) repository. The `cargo build-sbf --version` command shows rustc version used by the platform-tools.

   > If compilation fails because of incompatible dependencies, consider setting the concrete solana-program version in Cargo.toml: `solana-program = "=1.17.31"`.

# [Deploy](https://docs.solanalabs.com/cli/examples/deploy-a-program)

1. Make sure that the Solana Cluster is operational.
1. Airdrop some SOL to payer:
   1. `solana-keygen pubkey <PAYER KEYPAIR>`
   1. `solana airdrop 10 <PAYER PUBKEY>`
1. Run the`solana program deploy -ul --program-id <PROGRAM KEYPAIR> --keypair <PAYER KEYPAIR> <PROGRAM BINARY>` command to deploy to specific cluster. It will return the program id. Example:

   ```
   solana program deploy -ul --program-id ~/work/src/public/solana/programs/keypairs/ping.json --keypair ~/work/src/public/solana/programs/keypairs/payer.json ~/work/src/public/solana/programs/target/deploy/ping.so
   ```

   - Use the following command to redeploy: `solana program deploy -ul --program-id <PROGRAM PUBKEY> --upgrade-authority <PAYER KEYPAIR> <PROGRAM BINARY>`. Example:
     ```
     solana program deploy -ul --program-id HTnUFY3ogPkVuMoS3R6HnRDGRWg3Vi88WYHEHUu9XrNP --upgrade-authority ~/work/src/public/solana/programs/keypairs/payer.json ~/work/src/public/solana/programs/target/deploy/ping.so
     ```

# Run

- Use [Solana Clients](Solana-Development-Resources.md#solana-clients) to run on-chain programs.
- Make sure that the `solana_runtime::message_processor` logging is set to debug for seeing the program specific logging. Example:
  ```
  export RUST_LOG=solana=warn,solana_core=warn,solana_runtime=warn,solana_runtime::message_processor=debug,solana_streamer=warn,solana_poh=warn,solana_gossip=warn; solana-test-validator
  ```

# [CLI](https://docs.solanalabs.com/cli/examples/deploy-a-program)

- Show program information: `solana program show -u[m|t|d|l] <PROGRAM ID>`.
- Show all programs for given authority: `solana program show -um --programs --buffer-authority <PROGRAM AUTHORITY KEYPAIR>`.
